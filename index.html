Quick Start Guide

Getting Started:

  • Use Alt+N to quickly add new tasks
  • Double-click any task field to edit it inline
  • Click priority numbers (0-10) to change priority
  • Click status badges to cycle through states
  • Use search and filters to find specific tasks

Pro Tips

  • Column Management: Drag headers to reorder, double-click to rename
  • Sorting: Click column headers to sort data
  • Filtering: Use status checkboxes and priority dropdown
  • Resizing: Drag right edge of headers to resize
  • Custom Columns: Add columns with different data types
  • Data Import: Copy from Excel and paste in import dialog
